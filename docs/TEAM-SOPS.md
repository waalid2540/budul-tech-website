### Account Managers (Brothers)
- [ ] Check dashboard for new clients
- [ ] Call new clients within 24 hours of signup
- [ ] Complete onboarding checklist for each client
- [ ] Schedule monthly strategy calls
- [ ] Log all client interactions

### Content Manager (Child 1)
- [ ] Review AI-generated content for all clients
- [ ] Approve or request regeneration
- [ ] Schedule approved content
- [ ] Flag any issues to Account Manager

### Support (Child 2)
- [ ] Check support inbox every 2 hours
- [ ] Respond to all tickets within 4 hours
- [ ] Escalate complex issues to Account Manager
- [ ] Update FAQ if same question asked 3+ times

### Tech / Setup (Child 3)
- [ ] Check for new clients needing chatbot setup
- [ ] Install chatbot code on client websites
- [ ] Test chatbot is working
- [ ] Connect social media accounts
- [ ] Set up email/SMS integrations

## New Client Onboarding Checklist

### Day 1: Welcome
- [ ] Send welcome email
- [ ] Schedule onboarding call (within 48 hours)
- [ ] Add to client tracking spreadsheet

### Day 2-3: Onboarding Call (30 min)
- [ ] Introduce yourself
- [ ] Walk through dashboard
- [ ] Collect business info:
  - Business name, type, description
  - Website URL
  - Social media accounts
  - Target audience
  - Main services/products
- [ ] Set expectations (what we deliver, timeline)
- [ ] Schedule first strategy call (2 weeks out)

### Day 3-5: Setup
- [ ] Install AI chatbot on website
- [ ] Customize chatbot (name, greeting, colors)
- [ ] Connect social media accounts
- [ ] Create first week of content
- [ ] Set up lead notifications

### Day 7: Check-in
- [ ] Send "How's it going?" message
- [ ] Confirm chatbot is working
- [ ] Share first leads captured
